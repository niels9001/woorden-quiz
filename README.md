# 🎭 Woorden Quiz

Een vrolijke web-game om Nederlandse woordenschat (thema theater & film, Les 1 + Les 5) te oefenen op iPad, telefoon of computer.

## Hoe het werkt
- Je krijgt links een woord te zien
- Kies rechts de juiste omschrijving (4 opties)
- Vragen én antwoorden zijn elke keer in willekeurige volgorde
- Sterren, score, confetti en een highscore die wordt opgeslagen

## Speel hem
👉 https://niels9001.github.io/woorden-quiz/

## Lokaal draaien
Geen build nodig — gewoon `index.html` openen, of:

```bash
python -m http.server 8080
```

en open http://localhost:8080

## Bestanden
- `index.html` — markup
- `styles.css` — kleurrijk thema, responsive, animaties
- `app.js` — game logic
- `words.js` — woordenlijst (Les 1 + Les 5)
- `.github/workflows/pages.yml` — auto-deploy naar GitHub Pages
